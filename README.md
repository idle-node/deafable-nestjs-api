## Description

API for deafable project

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run db:dev:restart
$ yarn run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests (no unittest yet)
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
